## Getting Started

Install Bower

```sh
npm install -g bower
```

Install Polymer

```sh
npm install -g polymer-CLI
```

`cd` into this directory and install any missing bower dependencies with

```sh
bower install
```
finally you can serve locally with

```sh
polymer serve
```
